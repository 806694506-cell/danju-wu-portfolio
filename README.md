# 吴丹菊个人作品集网页

这是根据 `26届应届策划方向-吴丹菊.pdf` 制作的个人作品集静态网页草稿。

## 在线地址

https://806694506-cell.github.io/danju-wu-portfolio/

## 文件

- `index.html`：网页结构与内容。
- `styles.css`：黑底银白玻璃折射视觉风格，少量薄荷绿与蓝紫高光来自 `dark.js` 主题色。
- `script.js`：背景线条、视差、进场与卡片悬浮动效。
- `assets/avatar-comic.png`：漫画风格头像，用于公开网页。

## 注意

公开版不展示手机号，只保留邮箱联系方式。启动屏已嵌入 Spline 玻璃折射动效作为背景，主要文字由网页层控制。
